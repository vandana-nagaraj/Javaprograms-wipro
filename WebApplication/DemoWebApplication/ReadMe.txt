Servlet Context and Config
Listener class -- This is a special type of class that listens for application start up and shutdown events.
also if we want to set some application level variables (Context attributes )before any servlet runs.
listener class variables can be defined using annotation or web.xml <init-param>

Life Cycle of servlet:

submit ---- >  server  ----> Web container ( AddToCartServlet) ---- init()(it will be called once) ,service()(get , post ,put, delete), destroy()(it will be called once)

Context Parameters are defined centrally in a Listener class 
ServletContext ctx = sce.getServletContext();
    	ctx.setAttribute("discountRate" , 0.10); // 10% discount we are giving on products
    	ctx.setInitParameter("storeName"," ABC SHOPPING STORE"); // to store type of variable(application level)
    	System.out.println("Application context initialized");
    	
 To access 
 
     ServletContext context = getServletContext();
     String contextParamValue = context.getInitParameter("contextParameterName");
      
Config Parameter are defined in a servlet itself using annotation
@WebServlet(
		
		urlPatterns = "/WelcomeServlet",
		initParams = {
			@WebInitParam(name="welcomeMessage" , value=" Welcome to Online Shopping!")
		}
		)

  ServletConfig config = getServletConfig();

JSP Scripting Elements
Types of tags in JSP
1. Scriplet Tag  :  <%  %>  --- where in between you write your java code
2. Expression Tag : <%= %>  --- where in between you can write your java variables to access the value
3 .Declarative Tag : <%! %> --- Where you can declare the variables but implicit objects cannot be used 


Types of Scope 
1. application level
2. pageContext/ page Level
3. session level
4. request level
